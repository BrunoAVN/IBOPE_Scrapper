# IBOPE Scrapper

Neste repositório, implementamos um web scrapper para analisar os dados de audiência dos principais canais de TV aberta do Brasil ao longo dos anos.


## Construido com

*[Pandas](https://pandas.pydata.org/) - Biblioteca para organizar e analisar os dados coletados

*[Beautiful Soup 4](https://www.crummy.com/software/BeautifulSoup/bs4/doc/) - Biblioteca para analisar documentos HTML

*[Matplotlib](https://matplotlib.org/) - PAra visualização dos dados coletados
